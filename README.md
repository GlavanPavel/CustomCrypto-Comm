# CustomCrypto-Comm
Sistem distribuit de comunicare intre statii  (Socket (TCP/UDP) / Websocket, etc.), de transmitere point-to-point a mesajelor criptate  (demo intre 2-3 laptop-uri / statii / VMs) + fisiere criptate: criptare (+ impartire) + decriptare si refacere fisier Algoritm simetric de criptare - RC6 sau Blowfish
